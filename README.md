# Forge IDE

A modern, highly interactive IDE platform tailored for competitive programming, especially Codeforces.

## Chrome Extension Setup

To install the Forge IDE Chrome Extension:

1. `cd extension`
2. `npm install`
3. `npm run build`
4. Open Chrome and navigate to `chrome://extensions`
5. Enable "Developer mode" in the top right
6. Click "Load unpacked" and select the `extension/dist` folder

The extension allows you to open Codeforces problems directly in Forge IDE and submit your solutions back to Codeforces.
